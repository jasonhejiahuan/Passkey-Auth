Performance gate

遥测总开关

关闭后不读取遥测库、不修改 HTML、不加载采集脚本，也不创建网络任务。

默认采集能力

精确 CPU 型号不会被现代浏览器暴露；硬件摘要仅使用逻辑核心、粗粒度内存和可用的架构提示。字体与电池属于高级信号，建议只对指定用户开启。

设置更改后自动保存。采集会在浏览器空闲时运行，并按操作系统只下载匹配的模块。

Storage & delivery

接收端与发送路径

外部适配器只在被选中且收到有效样本时加载。密钥仅保存在服务端，不会回显到浏览器。

浏览器直连时，Passkey-Auth 只用 API Key 换取 v12 一次性采集地址，API Key 本身不会下发。

直连配置会被浏览器看到，因此不能使用服务端私有 Bearer/Header 密钥；自定义 Headers 也只能放公开元数据。
